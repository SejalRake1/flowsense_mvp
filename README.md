# FlowSense

### AI Expense Intelligence for UPI Users

FlowSense is an AI-powered MVP that helps users automatically track and understand their UPI spending habits — without manual expense entry or bank linking.

Built for India’s UPI-first generation.

## Features

* Auto-detects UPI transactions from notifications
* AI-generated spending insights
* Smart dashboard with categories & trends
* Detects subscriptions and money leaks
* Privacy-first architecture

## Tech Stack

* React Native + Expo
* Supabase
* OpenAI API
* PostgreSQL
* Expo EAS
